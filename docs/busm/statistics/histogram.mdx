### **Concept: Frequency Density and Histograms**

A **histogram** is a type of bar chart used to represent the distribution of continuous numerical data. Unlike a standard bar chart where the height of the bar shows the frequency, in a histogram, the **area of the bar** represents the frequency.

This becomes very important when the class intervals (the groups or "bins") have **unequal widths**. To ensure the graph is accurate and not misleading, we plot **Frequency Density** on the vertical (y) axis.

**The Rule:**
Frequency Density is calculated with the following formula:
$$\text{Frequency Density} = \frac{\text{Frequency}}{\text{Class Width}}$$
* **Frequency:** The number of data points in a class.
* **Class Width:** The difference between the upper and lower bounds of the class interval.

When drawing the histogram, the **width** of each bar is its class width, and the **height** is its frequency density. This way, the area of the bar (Height × Width) correctly represents the frequency.

### **Worked Examples from Past Papers**

:::danger 19th November 2024, Section B, Question 21
Complete the following table of frequency densities:
| Commute time, t (minutes) | Frequency |
| :--- | :---: |
| $0 \le t < 10$ | 6 |
| $10 \le t < 25$ | 33 |
| $25 \le t < 45$ | 50 |
| $45 \le t < 65$ | 72 |
| $65 \le t < 90$ | 55 |
| $90 \le t < 120$ | 24 |
:::
:::success Answer
| Commute time, t (minutes) | Frequency | Class Width | Frequency density |
| :--- | :---: | :---: | :---: |
| $0 \le t < 10$ | 6 | 10 | 0.6 |
| $10 \le t < 25$ | 33 | 15 | 2.2 |
| $25 \le t < 45$ | 50 | 20 | 2.5 |
| $45 \le t < 65$ | 72 | 20 | 3.6 |
| $65 \le t < 90$ | 55 | 25 | 2.2 |
| $90 \le t < 120$ | 24 | 30 | 0.8 |
:::note Explanation
1.  **Row 1:** Class Width = $10 - 0 = 10$. Frequency Density = $6 \div 10 = 0.6$.
2.  **Row 2:** Class Width = $25 - 10 = 15$. Frequency Density = $33 \div 15 = 2.2$.
3.  **Row 3:** Class Width = $45 - 25 = 20$. Frequency Density = $50 \div 20 = 2.5$.
4.  **Row 4:** Class Width = $65 - 45 = 20$. Frequency Density = $72 \div 20 = 3.6$.
5.  **Row 5:** Class Width = $90 - 65 = 25$. Frequency Density = $55 \div 25 = 2.2$.
6.  **Row 6:** Class Width = $120 - 90 = 30$. Frequency Density = $24 \div 30 = 0.8$.
